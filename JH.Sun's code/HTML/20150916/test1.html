提问者采纳 
首先，如果你要用ctrl+alt+f格式化代码的话要先选中一段代码再按才会有效，否则是无效的。
其次，如果你是要格式化整个html文件的话，可以从edit----Tag----Auto-Format Tags on Document.点击这个选项就可以格式化整个html文件。
如果有什么问题可以私信或加q，如果我知道的话会尽力帮你解答。
